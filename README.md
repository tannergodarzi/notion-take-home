# Installation
Hello friend - thank you for taking the time to review this code. Installation requires running `npm install` from the root folder.

# Instructions For Running
To run this project, please build the project, run the server and start the server environment using the commands below. 

- npm run build
- serve -s build

Or for the development server run
- npm run server
- npm run start

You should be able to view the project at http://localhost:3000 but `npm run start` will give you the correct URL.
